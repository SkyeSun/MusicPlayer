# 主要功能
 - 播放，暂停，切换音乐
 - 调节音乐进度
 - 切换播放模式(列表循环，列表随机)
 - CD 动画播放效果
 - 环形进度条显示
 - 点击列表播放相应歌曲
 
更多功能正在努力添加中...
